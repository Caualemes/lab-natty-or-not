# Avatar IA

## 📒 Descrição
O projeto consiste em um "clone" meu, gerado por IA's

## 🤖 Tecnologias Utilizadas
Funções Avatares e Clonar Voz, da plataforma Vidnoz

## 🧐 Processo de Criação
Enviando uma faixa de audio, uma foto e um script, a IA fez sua mágica
